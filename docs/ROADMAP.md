# Roadmap

This roadmap is intentionally small so each feature can become one focused pull request.

## Phase 1: Foundation

- Create the first playable tapper screen.
- Add optional local profile creation.
- Store profile and hit stats in browser storage.
- Document the GitHub and Codex workflow.

## Phase 2: Game Feel

- Add tap animations and short feedback states.
- Add milestones at 10, 50, 100, and 500 hits.
- Add upgrade buttons such as auto tap or bonus multiplier.
- Add a reset confirmation modal.

## Phase 3: Profiles

- Add editable profile names.
- Add multiple local profiles.
- Add simple player achievements.
- Prepare the data model for a future backend.

## Phase 4: Production Readiness

- Add tests for storage and stat calculations.
- Add GitHub Actions for lint and build checks.
- Add deployment instructions for GitHub Pages or Vercel.
- Add release notes and version tags.

## Suggested First Issues

- Add a visible milestone message when the player reaches 100 hits.
- Add an edit profile button.
- Add keyboard support so the Space key counts as a tap.
- Add a `CONTRIBUTING.md` checklist for new pull requests.
