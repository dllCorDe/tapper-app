# Product Notes

## App Concept

Tapper App is a tiny clicker game. The first version proves the main loop: choose a play mode, tap the big button, and watch the numbers grow.

## User Stories

- As a player, I can create a local profile so the app can remember me later.
- As a player, I can skip profile creation so I can play privately.
- As a player, I can tap a large button so each tap counts as a hit.
- As a player, I can see my total hits and streak stats.

## Current Scope

The app is frontend-only. It uses browser storage, not a backend account system. This keeps the project easy to run and review while leaving space for real profile functionality later.

## Future Ideas

- Multiple profiles
- Achievements
- Upgrades
- Leaderboards
- Backend authentication
- Cloud saves
